Generate a unique one-time download link for a file in the media library through WPForms.

== Description ==

This plugin allows users to create a one-time download link for files in the media library through WPForms. The link expires after being used once.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the plugin as described in the plugin's settings page.

== Frequently Asked Questions ==

= Does this plugin require WPForms to function? =
Yes, this plugin is designed to work alongside WPForms.

= How long does the download link last? =
The download link is one-time use. After being accessed once, it will no longer be valid.
